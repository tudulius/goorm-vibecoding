# 모던 책 소개 랜딩 페이지

`Day2_260430_목` 폴더에 구현된 정적 웹 프로젝트입니다.  
한 권의 책을 집중적으로 소개하는 모던/미니멀 콘셉트 랜딩 페이지를 HTML, CSS, JavaScript만으로 구성했습니다.

## 실행 방법

별도 빌드 없이 바로 실행할 수 있습니다.

1. `Day2_260430_목/` 폴더로 이동
2. `index.html` 파일을 브라우저에서 열기

또는 로컬 서버를 사용해 실행할 수 있습니다.

## 핵심 기능

- 단일 책 중심의 랜딩 구성
  - 히어로 섹션, 책 핵심 정보, 줄거리/핵심 메시지, 독자 리뷰, CTA
- 모던/미니멀 UI 디자인
  - 카드형 정보 블록, 여백 중심 레이아웃, 깔끔한 컬러 시스템
- 내비게이션 인터랙션
  - 앵커 기반 스무스 스크롤
  - 스크롤 위치에 따라 현재 섹션 메뉴 활성화
  - 모바일 메뉴 토글
- CTA 사용자 피드백
  - 구매 버튼 클릭 시 상태 문구 표시

## 파일 구조

- `Day2_260430_목/index.html`
  - 책 소개 랜딩 페이지의 전체 구조와 섹션 마크업
- `Day2_260430_목/style.css`
  - 레이아웃, 타이포그래피, 컴포넌트 스타일, 반응형 규칙
- `Day2_260430_목/script.js`
  - 내비게이션 활성화, 모바일 메뉴 열기/닫기, CTA 피드백 이벤트
