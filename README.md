# NEXTAI Studio (바이브코딩1기)

AI 에이전트 기반 앱 개발 에이전시 **NEXTAI Studio** 소개용 싱글 페이지 랜딩(HTML/CSS/JS, 프레임워크 미사용)입니다.

## 데모/실행 방법

이 프로젝트는 빌드 도구가 필요 없는 정적 웹페이지입니다.

1. `Day1_260429_수/` 폴더로 이동
2. `index.html`을 브라우저에서 열기

> 예) Finder/Explorer에서 `index.html` 더블클릭 또는 로컬 정적 서버로 제공

## 주요 기능

### 1) 라이트/다크 테마
- 화면 우측 상단의 **테마 토글**로 라이트/다크 전환
- 선택값은 `localStorage`에 저장됩니다.

### 2) 다국어 보기 (KO / EN / JA)
- 언어 선택 드롭다운에서 화면 텍스트와 입력 placeholder가 변경됩니다.
- `data-i18n` 키 기반으로 자동 매핑합니다.

### 3) 고객 문의 폼 (하단 CTA 영역)
- 이름/이메일/회사(선택)/문의사항 입력 후 전송
- 전송은 `mailto:` 기반으로 동작하며, 사용자의 메일 클라이언트가 열립니다.

수신 이메일 주소는 `Day1_260429_수/script.js` 상단의 `toEmail` 값을 수정하세요.

## 파일 구조

- `Day1_260429_수/index.html` : 페이지 마크업(싱글 페이지)
- `Day1_260429_수/styles.css` : 라이트/다크 테마 + 컴포넌트 스타일
- `Day1_260429_수/script.js` : 테마/언어 토글, FAQ 아코디언, 문의 폼 처리, CTA 트래킹 훅
- (참고) `Day1_260429_수/style.css` : 기존/레거시 스타일 파일(현재 `index.html`은 `styles.css`를 사용)
